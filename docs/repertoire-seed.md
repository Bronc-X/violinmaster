# ViolinMaster MVP Repertoire Seed Library

This seed library is for product planning and metadata. The MVP should only bundle full scores/audio when the asset is public domain, openly licensed, self-authored, or provided by the project owner. Copyrighted Suzuki, exam, and Chinese repertoire can still appear as catalog metadata and import targets.

## Selection Principles

- Cover beginner to advanced violin learners, roughly Suzuki Book 1 through conservatory/exam Level 10.
- Prefer pieces with clear MusicXML/MIDI availability for the first playable MVP.
- Include technique tags so the analysis agent can connect audio errors with likely practice problems.
- Keep copyrighted pieces as metadata-first entries until usable score/audio assets are supplied.

## Technique Tag Legend

- `intonation`: pitch accuracy and hand-frame stability
- `rhythm`: pulse, subdivision, entrance timing
- `shifting`: position changes
- `fourth-finger`: fourth-finger stability and hand spacing
- `bowing`: bow direction, slurs, string crossings, articulation
- `vibrato`: vibrato consistency and control
- `double-stops`: two-note tuning
- `fast-passages`: passagework, runs, ornaments
- `tone`: sustained sound, contact point, bow speed
